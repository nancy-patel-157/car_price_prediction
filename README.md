# Car Price Prediction using Machine Learning

A Machine Learning web application that predicts the selling price of a used car based on user inputs. The project uses **Linear Regression** for prediction and **Flask** to provide an interactive web interface.

---

##  Project Overview

This project helps users estimate the price of a used car by entering details such as the car company, model, year, fuel type, and kilometers driven.

The application performs preprocessing on the input data, loads a trained Machine Learning model, and returns the predicted selling price instantly.

---

##  Features

- Predicts used car prices in real time
- User-friendly web interface
- Machine Learning model trained using Linear Regression
- Data preprocessing and feature engineering
- Flask backend for serving predictions
- Ready for deployment

Dataset includes:

- Company

- Kilometers Driven
- Selling Price

##  Installation

### Clone the repository

```bash
git clone https://github.com/nancy-patel-157/car_price_prediction.git
```

Move into the project directory

```bash
cd car_price_prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000/
```
